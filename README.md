# “Exploratory Data Analysis of Electronics Sales using SQL and Python”

- SQL Data Cleaning: Designed queries to restructure messy fields (e.g., extracting Average_Rating, Reviews_Count, BrandName) and normalize price/discount columns for accurate reporting.

- Exploratory Analysis: Computed descriptive statistics (min, max, mean, std) and created categorical distributions (histograms, pie charts) to uncover pricing and product trends.

- Business Insights: Identified sub‑category distributions and discount patterns, enabling clear visualization of product segments and customer review behaviors.

- Feature Engineering: Converted raw text fields into structured numerical features (ratings, counts, discount flags), preparing the dataset for predictive modeling.

- Statistical Analysis: Applied covariance calculations between Reviews_Count and Average_Rating to explore relationships and potential predictors of customer satisfaction.

- Visualization for Modeling: Built scatter plots and distribution charts to validate assumptions, spot outliers, and assess variable correlations before model training.
